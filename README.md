# Пет проект "Доставка еды"
	Доставка еды(Flutter pet project)


## Описание
 приложение для доставки еды 

Стэк использованных технологий:
- Dart/Flutter
- Provider
- Firebase Storage
- Firebase Auth
- FireStore Database


## Установка
 Инструкции по установке проекта. Введите следующие команды в терминал
 1. flutter pub get
 2. flutter run
